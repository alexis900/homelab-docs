# MNT — Actualización UniFi Network Application 10.0.162 → 10.1.83

## Descripción

Actualizar la **aplicación UniFi Network** desde la versión *10.0.162* a la *10.1.83* para obtener mejoras, correcciones y nuevas funcionalidades incluidos en esta actualización.

> Nota: Se recomienda realizar siempre copia de seguridad completa antes de actualizar.

---

## Sistemas afectados

- Controlador UniFi Network (servidor/autoinstalado o en consola UniFi OS)
- Consola de administración web de UniFi Network

---

## Acciones realizadas

- [ ] Copia de seguridad completa de la configuración de UniFi Network
- [ ] Descargar instalador versión 10.1.83
- [ ] Detener el servicio de UniFi Network antes de la actualización
- [ ] Ejecutar actualización mediante instalador o vía repositorio/apt
- [ ] Reiniciar el servicio UniFi Network
- [ ] Comprobar que la versión correcta está corriendo

---

## Validaciones

- [ ] Servicio **UniFi Network operativo** tras la actualización
- [ ] **Versión confirmada 10.1.83** en la interfaz de administración
- [ ] **Logs sin errores críticos** relevantes
- [ ] **Conectividad de dispositivos UniFi** intacta y adoptados correctamente
