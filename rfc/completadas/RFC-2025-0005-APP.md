# RFC — Despliegue de Proxmox Backup Server (PBS) como VM en Apollo

## Resumen

Se desplegó una instancia de **Proxmox Backup Server (PBS)** como máquina virtual en el nodo **Apollo**, con almacenamiento dedicado y accesible desde Zeus. PBS centraliza y protege las copias de seguridad de nodos, CTs y VMs del entorno Proxmox, asegurando disponibilidad y consistencia ante desastres.

---

## Motivación

- Disponer de un sistema de copias de seguridad centralizado, seguro y gestionado desde el ecosistema Proxmox.  
- Evitar soluciones externas y facilitar automatización de snapshots, backups y restauraciones.  

---

## Justificación de criticidad

PBS será el sistema encargado de centralizar y proteger todas las copias de seguridad de los nodos, contenedores y VMs del entorno Proxmox. Su disponibilidad y consistencia son **críticas** para la recuperación ante desastres.

---

## Dispositivo(s) / servicio(s) afectados

- Nodo Proxmox: Apollo  
- VM: pbs.home.arpa  
- VLAN: 99 (gestión)  
- Almacenamiento PBS: dataset dedicado o disco virtual `pbs`

---

## Plan de acción

1. Crear un dataset ZFS en Apollo: `zfs-pbs`.  
2. Crear VM con:  
   - Nombre: `pbs.home.arpa`  
   - 4 vCPU, 3 GB RAM, 100 GB disco (ajustable)  
   - Red en VLAN 99  
3. Instalar Proxmox Backup Server (última versión estable).  
4. Configurar certificados TLS con CA interna.  
5. Integrar PBS en el clúster Proxmox.  
6. Crear backup jobs para:  
   - VMs y CTs de Apollo  
   - Futuras VMs/CTs de Zeus  
7. Documentar IP, acceso y credenciales.  
8. Probar restauración de un CT o snapshot.

---

## Riesgos

- Consumo de recursos en Apollo.  
- Posible sobrecarga del nodo si hay múltiples tareas concurrentes.  
- Pérdida de datos si no se configura correctamente almacenamiento o ciclos de retención.

---

## Mitigación

- Establecer límites de tareas simultáneas.  
- Monitorización con Uptime Kuma.  
- Replicación futura en nodo baremetal (cuando esté disponible).  
- Verificaciones periódicas de backups (checks de integridad).  
- Guardar snapshots críticos también en almacenamiento externo si se habilita.

---

## Resultado

Instalación completada satisfactoriamente. PBS se encuentra operativo como VM en Apollo, con backups funcionando correctamente y accesible para restauraciones de prueba.
