## Título:

Despliegue de Proxmox Backup Server (PBS) como VM en Apollo

## Tipo de cambio:

SRV

## Fecha propuesta:

2025-11-15

## Estado:

Propuesta

## Criticidad:

Alta

## Justificación de criticidad:

PBS será el sistema encargado de centralizar y proteger todas las copias
de seguridad de los nodos, contenedores y VMs del entorno Proxmox. Su
disponibilidad y consistencia son críticas para la recuperación ante
desastres.

## Resumen:

Se propone desplegar una instancia de Proxmox Backup Server como máquina
virtual en el nodo Apollo, con almacenamiento dedicado y accesible desde
Zeus. A futuro, está prevista su migración a un nodo baremetal dedicado
cuando se disponga de uno.

## Motivación:

Disponer de un sistema de copias de seguridad centralizado, seguro y
gestionado desde el propio ecosistema Proxmox, evitando soluciones
externas y facilitando la automatización de snapshots, backups y
restauraciones.

## Dispositivo(s) / servicio(s) afectados:

\- Nodo Proxmox: Apollo\
- VM: pbs.home.arpa\
- VLAN: 99 (gestión)\
- Almacenamiento PBS: pbs (dataset dedicado o disco virtual)

## Plan de acción:

1\. Crear un nuevo dataset ZFS en Apollo: zfs-pbs.\
2. Crear VM con:\
- Nombre: pbs.home.arpa\
- 4 vCPU, 3 GB RAM, 100 GB de disco (ajustable).\
- Red en VLAN 99.\
3. Instalar Proxmox Backup Server (última versión estable).\
4. Configurar certificados TLS con CA interna.\
5. Integrar PBS en el clúster Proxmox.\
6. Crear backup jobs para:\
- VMs y CTs de Apollo.\
- En el futuro: VMs/CTs de Zeus.\
7. Documentar IP, acceso y credenciales.\
8. Probar restauración de un CT o snapshot.

## Riesgos:

\- Consumo de recursos en Apollo.\
- Posible sobrecarga del nodo si hay múltiples tareas concurrentes.\
- Pérdida de datos si no se configura correctamente el almacenamiento o
los ciclos de retención.

## Mitigación:

\- Establecer límites de tareas simultáneas.\
- Monitorización con Uptime Kuma.\
- Replicación futura en nodo baremetal (cuando esté disponible).\
- Verificaciones periódicas de backups (checks de integridad).\
- Guardar snapshots críticos también en almacenamiento externo si se
habilita.

## Resultado:

Completado de manera satisfactoria. Se ha instalado PBS como máquina
virtual en VirtualBox para hacer copias de seguridad.
