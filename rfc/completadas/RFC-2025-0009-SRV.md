# RFC-ID:

RFC-2025-0009-SRV

# Título:

Instalación de Nginx Proxy Manager (NPM) en VLAN 20

# Tipo de cambio:

SRV / NET

# Fecha propuesta:

2025-08-02

# Estado:

Completado

# Criticidad:

Media

# Justificación de criticidad:

NPM gestionará el proxy inverso para servicios críticos, por lo que una
instalación incorrecta o fallos en la configuración podrían afectar
accesos y disponibilidad.

# Resumen:

Despliegue de Nginx Proxy Manager en un contenedor LXC dentro de la VLAN
20 para gestionar accesos externos e internos mediante proxy inverso con
certificados TLS.

# Motivación:

Centralizar la gestión de proxy inverso para simplificar configuraciones
SSL, redirecciones y acceso seguro a servicios internos.

# Dispositivo(s) afectados:

-   Contenedor LXC en nodo Proxmox Apollo

-   VLAN 20

-   Servicios internos accesibles vía proxy

# Plan de acción:

-   Completar previamente el RFC-2025-0008-NET para la creación y
    configuración de VLAN 20.

-   Crear contenedor LXC en VLAN 20 con IP fija (ej. 10.0.20.10).

-   Instalar Docker y Docker Compose.\
    Desplegar Nginx Proxy Manager usando docker-compose con
    configuración básica.

-   Configurar firewall para permitir acceso HTTP/HTTPS a NPM.

-   Acceder a la interfaz web y configurar usuario administrador.

-   Configurar certificados TLS (usando Let\'s Encrypt).

-   Configurar reglas de proxy inverso para servicios existentes.

-   Documentar configuración y credenciales.

# Riesgos:

-   Problemas de seguridad si no se configura adecuadamente el acceso a
    la interfaz.

-   Fallos en renovación automática de certificados TLS.

# Mitigación:

-   Verificar puertos disponibles antes de instalación.

-   Restringir acceso a la interfaz de administración por IP o VPN.

-   Monitorizar logs y estado de certificados.

# **Rollback:**

-   En caso de fallo, detener y eliminar el contenedor LXC de NPM.

-   Restablecer accesos directos temporales a servicios sin proxy.

-   Revisar configuración y reintentar despliegue.

# **Dependencias:**

-   Este cambio depende de la finalización exitosa de
    **RFC-2025-0008-NET** (Creación de la VLAN 20- DMZ para servicios
    expuestos).

# Resultado:

Instalación de Nginx Proxy Manager (NPM) en VLAN 20 completada con
éxito. El servicio está operativo y configurado para gestionar el proxy
inverso según lo planificado, sin incidencias durante la implementación.
