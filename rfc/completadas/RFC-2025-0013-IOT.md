# RFC-ID:

RFC-2025-0013-IOT

# Título:

Implementación de Zigbee2MQTT + Mosquitto en Home Assistant

# Tipo de cambio:

IOT

# Fecha propuesta:

2025-08-08

# Estado:

Completado

# Criticidad:

Alta

# Justificación de criticidad:

La integración de Zigbee2MQTT y Mosquitto es fundamental para la
comunicación estable y segura de dispositivos IoT en el sistema
domótico. Cualquier fallo puede afectar la automatización y
monitorización de sensores críticos en el hogar.

# Resumen:

Se implementaron los servicios Zigbee2MQTT y Mosquitto dentro de Home
Assistant para gestionar dispositivos Zigbee mediante MQTT, mejorando la
interoperabilidad y el control centralizado.

# Motivación:

Centralizar el control y monitorización de dispositivos Zigbee a través
de MQTT, facilitando la integración y el mantenimiento del sistema
domótico.

# Dispositivo(s) afectados:

-   Servidor Home Assistant (HA)

-   Dispositivos Zigbee compatibles con Zigbee2MQTT

-   Broker MQTT Mosquitto

-   VLAN 40 - Dispositivos IoT

# Plan de acción:

-   Preparar entorno de Home Assistant para la integración.

-   Instalar y configurar Zigbee2MQTT para detección y gestión de
    dispositivos Zigbee.

-   Instalar y configurar Mosquitto como broker MQTT local.

-   Integrar Zigbee2MQTT con Mosquitto y Home Assistant.

-   Realizar pruebas de comunicación y control de dispositivos Zigbee.

-   Documentar configuración y procedimientos.

# Riesgos:

-   Pérdida de comunicación con dispositivos Zigbee durante la
    migración.

-   Conflictos en el broker MQTT si existen otros servicios similares.

-   Posible inestabilidad temporal en Home Assistant.

# Mitigación:

-   Realizar backups completos de Home Assistant antes de iniciar.

-   Implementar la configuración en entorno de pruebas previo a
    producción.

-   Monitorizar la red Zigbee y los logs del broker durante la
    implementación.

-   Mantener un plan de rollback rápido en caso de fallo.

# Resultado:

Implementación de Zigbee2MQTT y Mosquitto en Home Assistant (HA)
completada con éxito. Los dispositivos Zigbee se integraron
correctamente y la comunicación MQTT funciona establemente. No se
reportaron incidencias y el sistema está operando con alta
disponibilidad según lo esperado.
