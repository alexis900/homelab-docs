# RFC -- Reducción de memoria asignada a VM OPNsense

RFC-2025-0023-INFRA

**Título:** Reducción de memoria RAM asignada a la VM OPNsense en
Proxmox

**Tipo de cambio:** INFRA

**Fecha propuesta:** 2025-12-23

**Estado:** Completada

**Criticidad:** Baja

# Justificación de criticidad:

Cambio no disruptivo, reversible y sin impacto previsto en la
disponibilidad del servicio. La VM OPNsense presenta un consumo de
memoria estable y bajo, sin uso de servicios de inspección profunda
(IDS/IPS).

# Resumen:

Reducción de la memoria RAM asignada a la máquina virtual OPNsense de
3,5 GB a 2 GB para optimizar el uso de recursos del host Proxmox.

# Motivación:

El consumo real de memoria de OPNsense es de aproximadamente 500--600
MB. La asignación previa sobredimensionada provocaba una reserva
innecesaria de RAM en Proxmox, limitando recursos disponibles para otras
VMs.

# Dispositivo(s) afectados:

-   VM OPNsense

-   Host Proxmox (liberación de memoria)

# Plan de acción:

-   Modificar la asignación de memoria RAM de 3,5 GB a 2 GB en Proxmox.

-   Reiniciar la VM.

-   Verificar correcto funcionamiento de firewall, NAT, DNS y VPN.

-   Monitorizar uso de memoria y swap durante el funcionamiento normal.

# Riesgos:

-   Posible presión de memoria en caso de picos de carga inesperados.

# Mitigación:

-   Monitorización posterior del uso de RAM y swap.

-   Posibilidad de rollback inmediato aumentando de nuevo la memoria
    asignada.

# Resultado:

Tras el cambio, el consumo de memoria observado es de aproximadamente
461 MB.\
No se detectan errores, uso de swap ni impacto en el servicio. Cambio
validado.
